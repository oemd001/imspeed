## Table 1: Pre-Requisites

| Requirement          | Description                                                                                                                                        |
|----------------------|----------------------------------------------------------------------------------------------------------------------------------------------------|
| **Brio Deal ID**     | Needed when **GS is the lead bank** to identify the deal in Brio (passed as `ecmDealId`).                                                          |
| **CMG Offering ID**  | Needed when **GS is not the lead bank** to identify the offering in CMG (passed as `cmgOfferingId`).                                               |
| **Allocation Set ID**| Required in the non-lead scenario to specify which allocation set is reflowed from CMG to Brio (passed as `allocationSetId`).                      |
| **Swagger Endpoints**| - **Lead scenario** uses [uploadAllocationQuantitiesToCmg](https://cmgint.brio.fg.url.gs.com/swagger-ui/index.html#/allocation-facade/uploadAllocationQuantitiesToCmg).<br>- **Non-lead scenario** uses [reflowAllocationsSetToBrio](https://cmgint.brio.fg.url.gs.com/swagger-ui/index.html#/allocation-admin-facade/reflowAllocationsSetToBrio). |
